# Release notes — 10.2.0-alpha11

Overview

This release contains the porting progress and improvements in the kotlin-multiplatform `lucene-kmp` module since `10.2.0-alpha10`. 

Notable Milestones

- New Modules partially ported: queries
- Total Test Coverage: 73% (based on Kover and codecov)
- Among production classes of analysis, codecs, core, queries, queryparser, test-frameworks 1574 classes ported among 2399 (65%)
- Among unit test classes of those modules, 720 classes ported among 1186 (60%)
- In core module, test classes in package index, 118 classes ported among 197 (59%)
- In core module, test classes in package search, 104 classes ported among 149 (69%)
- Many bug fixed such as synchronization related bugs in IndexWriter, the codebase got more stable
